## About Laravel

1. Run ``composer install`` in console
2. Copy ``.env.example`` to ``.env``
3. Run ``php artisan key:generate`` in console
4. Open ```.env``` and update your db config
5. Run ``php artisan migrate`` in console
6. Run ``php artisan serve`` in console
7. Open browser and go to ``localhost:8000``
